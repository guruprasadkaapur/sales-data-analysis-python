# 📊 Sales Data Analysis (Python + Pandas)

## 🔍 Overview
This project analyzes retail sales data to uncover trends, patterns, and business insights using Python.

---

## 🧹 Data Cleaning
- Removed duplicate records
- Handled missing values
- Ensured consistent data formatting

---

## 📊 Analysis Performed
- Total Sales and Profit analysis
- Sales by Region
- Profit by Category
- Top-performing products

---

## 📈 Key Insights
- West region generates highest sales
- Technology category contributes most to profit
- Some products have high sales but low profit
- Revenue is concentrated among top-performing products

---

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
